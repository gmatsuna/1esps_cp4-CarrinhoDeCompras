Projeto desenvolvido para a disciplina **Web Development with JavaScript**.

O objetivo deste projeto é criar um **mini e-commerce interativo** utilizando **HTML, CSS e JavaScript**, aplicando conceitos de manipulação do DOM, armazenamento local e carregamento de dados via JSON.

---

# 📌 Descrição do Projeto

O **EcoTrend** é um e-commerce focado em produtos sustentáveis e ecológicos, incentivando um estilo de vida mais consciente.

O sistema permite que o usuário:

* visualize produtos
* filtre produtos por categoria
* adicione produtos ao carrinho
* remova produtos do carrinho
* limpe o carrinho
* finalize a compra

Todas as interações são feitas dinamicamente utilizando **JavaScript**, sem recarregar a página.

---

# 🚀 Funcionalidades

### Lista de produtos dinâmica

Os produtos são carregados a partir de um arquivo **JSON** e exibidos dinamicamente na página.

### Filtro de produtos

O usuário pode filtrar os produtos por categoria:

* Roupas e acessórios
* Produtos de beleza
* Itens para casa
* Tecnologia

### Carrinho de compras

O carrinho permite:

* adicionar produtos
* remover produtos
* limpar o carrinho
* visualizar o total da compra

### Persistência de dados

O carrinho é armazenado utilizando **localStorage**, permitindo que os itens permaneçam salvos mesmo após atualizar ou fechar o navegador.

### Finalização de compra

Ao finalizar a compra:

* o valor total da compra é exibido
* o carrinho é automaticamente limpo
